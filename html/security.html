SECURITY

Trust is our value.

Today every employee at CashRoot continues our founders' focus on security and compliance, in order to build trust. CashRoot has uilt a leading security program focused on developing innivative security solutions to help protect and secure our customers and their assets. We have also invested considerable resources to remain transparent about our security posture, through third party security assessments.

Reporting Fraud or Abuse

We have built a rulesbased marketplace with security at its core, so that our customers can focus on building their crypto portfolio. Our team is ready to help should you encounter fraud or abuse. You are very free to report fraud whenever you spot scams.

Coordinated Disclosure Program

We welcome contributions from security researchers to help us build and secure the future of money. Tosubmit a security vulnerability to CashRoot.

Valid CashRoot Communications

CashRoot does not offer phone support. All support is provided via email. You will only be protected via a valid CashRoot email. Emails from CashRoot will always end with the .cashroot.com domain name. For example, emails from support@cashroot.com are valid CashRoot addresses, because they end with .cashroot.com
